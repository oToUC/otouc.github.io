canvas-图形变换2

图形变换

canvas的图形变换函数是叠加的

  • 记得逆转~translate(100, 100)→translate(-100, -100)→translate(300, 300)
  • 但这样实在是太麻烦了，，
  • 因此save(), restore()出马了～～

Canvas状态的保存与恢复
save()
restore()
当我们绘制一个整体的元素时，特别是用到图形变换时注意使用save()与restore()
来保证canvas图形状态的绘制正确

应用translate, rotate & scale

scale()函数会有副作用，坐标，线条宽度和本身都会scale

变换矩阵
a c e
b d f
0 0 1

1 0 0
0 1 0
0 0 1

a 水平缩放(1)
b 水平倾斜(0)
c 垂直倾斜(0)
d 垂直缩放(1)
e 水平位移(0)
f 垂直位移(0)

a d 缩放
b c 倾斜
e f 位移
transform(a, b, c, d, e, f);
默认
transform(1, 0, 0, 1, 0, 0);
