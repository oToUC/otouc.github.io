canvas-画弧和圆

{ draw: arc }

画弧其实我们可以很容易联想到以下几点？（弧是圆中的一部分）

  1. 知道圆的中心点(x, y)
  2. 知道圆的半径
  3. 知道弧在圆上的开始于某个弧度
  4. 知道弧在圆上的结束于某个弧度
  5. 是顺时针画，还是逆时针画
  • 因此在canvas中的对应的arc方法参数也就不言而喻了
  • 实际演练
  1. beginPath()指的是重新规划一条路径
  2. closePath()指的是要结束当前的路径，如果当前的路径没有封闭的话，会自动让当前的路径封闭
  3. beginPath() 与 closePath()不一定要成对出现
  4. 对于context.fill(), closePath()写与不写都是先把路径封闭，然后再填充

canvas系列之初体验①

Step 1

css样式设置画布的宽高只是画布的显示宽高

  1. html

  2. javascript

Step 2

canvas中绘图是一种基于状态的绘图,绘图过程中先设置绘图的状态，然后调用具体的函数做具体的绘制

  1. 画一条直线

  2. 绘制一个直角三角形

  3. 填充多边形

  4. 填充+边框（它们的先后顺序会产生不同的效果）

  5. 想画一个蓝色的三角形和一条黑色线段，结果发现线条的颜色都变成了黑色。原因是canvas绘制是基于状态的绘制

  6. 如何分开这两个路径绘制的装态?当然是context.beginPath()与context.closePath()

Summary

  • Draw

  1. 首先，我们可以用moveTo和lineTo两个方法来定义一条路径。
    如果想要把多条路径隔开处理，就需要用beginPath与closePath来包裹路径。

  2. lineWidth→定义线条的粗细
    strokeStyle→定义线条的颜色
    fillStyle→定义路径所包裹区块的填充颜色
    stroke()→线条的具体绘制
    fill()→路径所包裹区块的具体绘制
