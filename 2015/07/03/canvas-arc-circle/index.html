canvas-画弧和圆

{ draw: arc }

画弧其实我们可以很容易联想到以下几点？（弧是圆中的一部分）

  1. 知道圆的中心点(x, y)
  2. 知道圆的半径
  3. 知道弧在圆上的开始于某个弧度
  4. 知道弧在圆上的结束于某个弧度
  5. 是顺时针画，还是逆时针画
  • 因此在canvas中的对应的arc方法参数也就不言而喻了
  • 实际演练
  1. beginPath()指的是重新规划一条路径
  2. closePath()指的是要结束当前的路径，如果当前的路径没有封闭的话，会自动让当前的路径封闭
  3. beginPath() 与 closePath()不一定要成对出现
  4. 对于context.fill(), closePath()写与不写都是先把路径封闭，然后再填充
